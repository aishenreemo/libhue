### libhue
A lightweight C library that is aimed to simplify color management in C programs, offering essential utility functions and macros for effortless color handling.

#### installing
```zsh
make
sudo make install
```

#### uninstalling
```zsh
sudo make uninstall
```

#### usage
Put `-lhue` flag on your compiler then
```c
#include <hue.h>

// your code
```
